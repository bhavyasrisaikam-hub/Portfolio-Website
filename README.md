# Advanced CSS3 Responsive Portfolio Website

## Objective

To develop a fully responsive personal portfolio website using HTML5, CSS3, JavaScript, CSS Grid, and Flexbox.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* CSS Grid
* Flexbox
* VS Code
* GitHub Pages

## Features

* Responsive Design
* Dark Mode Toggle
* Skills Section
* Projects Showcase
* Contact Form
* Modern User Interface
